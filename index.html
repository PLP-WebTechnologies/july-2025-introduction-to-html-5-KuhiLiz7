About This Page

This web page demonstrates how to create a basic website using and add semantic elements to improve accessibilty and SEO.

Key Features

Semantic Structure

We use elements like:

  • 'header' - for introducing the Web Page
  • 'footer' - indicating an end to your Web Page
  • 'nav' - to allow user to access different services featured in the Web Page
The elements should be enclosed in tags where there is the opening tag, content, then the closing tag.

Accessibility

Elements include ARIA roles and labels where necessary to help users interpret content correctly.
